What is Object-Oriented Programming (OOP)?
A. A programming paradigm based on functions
B. A programming paradigm based on objects and classes
C. A method of writing procedural code
D. A debugging technique
ANSWER: B

Which keyword is used to define a class in Python?
A. class
B. def
C. struct
D. object
ANSWER: A

What is a class in Python?
A. A built-in function
B. A blueprint for creating objects
C. A collection of data
D. A variable storing multiple values
ANSWER: B

How do you create an object from a class named `Car`?
A. obj = Car()
B. obj = new Car()
C. obj.create(Car)
D. obj = Car.new()
ANSWER: A

Which method is used as a constructor in Python classes?
A. __construct__()
B. __start__()

Which keyword is used to refer to the current instance of a class in Python?
A. this
